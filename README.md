# Loup-Garou-Solo-Phone

EN/
A digital version of the Werewolf game where players take turns with a single device to discover roles, make decisions, and try to survive or eliminate others.

GAME PRESENTATION :
This game is a digital adaptation of the classic *Werewolf of Thiercelieux*. It is designed to let players enjoy the game using a single smartphone, making it easy to play anywhere.

Additional mechanics have been added to ensure smooth gameplay and a proper game flow for all players.

To access the game, simply enter the following URL in your browser:
https://taz-off.github.io/Loup-Garou-Solo-Phone/

HOW TO USE THE GAME OFFLINE :
This game can be used without an internet connection once it has been installed on your device.
First, open the game in your browser using the provided link. Then, add it to your home screen (on mobile) or install it as an app if your browser allows it. This process saves the game locally on your device.
After installation, you can launch the game directly from your home screen without needing internet access. All game features, including role selection and guided gameplay, will work offline.
Make sure to open the game at least once while connected to the internet so that all files are properly loaded and stored on your device.



# Loup-Garou-Solo-Phone

FR/
Une version numérique du jeu du Loup-Garou où les joueurs se passent un seul appareil pour découvrir leur rôle, prendre des décisions et tenter de survivre ou d’éliminer les autres.

PRÉSENTATION DU JEU :
Ce jeu est une adaptation numérique du célèbre *Loup-Garou de Thiercelieux*. Il est conçu pour permettre aux joueurs de jouer à l’aide d’un seul smartphone, ce qui le rend facile à utiliser partout.

Des mécaniques supplémentaires ont été ajoutées afin d’assurer une expérience fluide et un bon déroulement de la partie pour tous les joueurs.

Pour accéder au jeu, il suffit de saisir l’URL suivante dans votre navigateur :
https://taz-off.github.io/Loup-Garou-Solo-Phone/

COMMENT UTILISER LE JEU HORS LIGNE :
Ce jeu peut être utilisé sans connexion internet une fois installé sur votre appareil.
Commencez par ouvrir le jeu dans votre navigateur à l’aide du lien fourni. Ensuite, ajoutez-le à votre écran d’accueil (sur téléphone) ou installez-le comme une application si votre navigateur le permet. Cette étape enregistre le jeu localement sur votre appareil.
Après l’installation, vous pouvez lancer le jeu directement depuis votre écran d’accueil sans connexion internet. Toutes les fonctionnalités, y compris le choix des rôles et le déroulement guidé de la partie, fonctionneront hors ligne.
Assurez-vous d’ouvrir le jeu au moins une fois avec une connexion internet afin que tous les fichiers soient correctement chargés et enregistrés sur votre appareil.
